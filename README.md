# TNTdle - игра-угадайка по сериалам с телеканала ТНТ

## Инструкция пользователя:
1. "Угадать по характеристикам":
   - Введите имя персонажа в поле ввода/выберите из выпадающего списка, чтобы приблизиться к разгадке случайно загаданного персонажа по совпадению характеристик
2. "Угадать по цитате":
   - Прочтите цитату и введите имя персонажа в поле ввода/выберите из выпадающего списка, чтобы угадать или послушать цитату загаданного персонажа
3. "Все персонажи"/кнопка вопроса в окне игры:
   - В отдельном окне вам будут показаны все персонажи, который могут быть загаданы в данной игре

## Не забудьте установить зависимости перед началом игры:
```bash
    pip install -r requirements.txt 
```
